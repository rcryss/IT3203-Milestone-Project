What is HTTP?

Hypertext Transfer Protocol (HTTP) is one of the fundamental building blocks of the World Wide Web. Developed by Tim Berners-lee in 1989, HTTP was designed to facilitate communication between web browsers and servers. HTTP is an Application layer protocol in the OSI model, meaning that it is responsible for providing services that applications use to communicate over a network. HTTP operates on a Client-server model, where clients send requests and servers return responses. This website explores how these protocols work, and how they have evolved over time, and remain key essentials in modern day web communication.
