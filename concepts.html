Key Concepts

Term Definition
World Wide Web World Wide Web is a collection of websites and resources that can be accessed through the internet by users around the world.
HTTP HyperText Transfer Protocol (HTTP) is an application layer protocol that enables communication between clients and servers in order to exchange information over the internet.
HTTPS HyperText Transfer Protocol Secure (HTTPS) is a secure version of HTTP which encrypts the data being transferred.
HTML HTML is a markup language utilized to create the structure content on websites.
Client Client is a device such as a computer that sends a request to a server.
Server A server is system that listens for client requests and responds with the requested information
POST The POST method is an HTTP request that sends data to a server to update a resource.
GET GET requests the data from a server
PUT PUT replaces an entire resource or creates a new one if it doesn't exist
PATCH PATCH is used to modify an existing resource
DELETE DELETE permanently deletes a resource from a server
